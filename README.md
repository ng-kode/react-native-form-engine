# Features

- Easy to configure templates (for both inputs and displays)
- Full control of `value` and `onChange`
- Handled `errors` and `touched` for you
- Disable / Hide fields by conditions

# Installation

```
$ yarn install
$ cd ios && pod install
$ cd .. && react-native run-ios
```
